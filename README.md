## Why use html?
### The main use of HTML is to build structure and organize content on a web page. Without HTML, browsers would not know how to display text, images, or other elements in a structured manner.Main Uses of HTMLCreating Page StructureHTML divides web pages into clear sections. Examples include the head (header), main content (body), and footer (footer). Displaying Text Content HTML sets the basic text format. This determines which text functions as a title (heading), paragraph, or list of points (list). Inserting MediaHTML is used to insert visual and audio elements. You can display images, play videos, or install a sound player directly on a web page. Creating Navigation (Links) HTML connects one page to another page via hyperlinks. This is the basic function that makes the internet interconnected.Creating Interactive FormsHTML provides elements for collecting data from visitors. Examples are text fields, selection buttons, and submit buttons for contact or login forms. Helps SEO (Search Engine Optimization)HTML uses special tags that help search engines like Google understand the content of your content. A good HTML structure makes the website easier for other people to find.

## What html it is?
### HTML (HyperText Markup Language) is a standard language used to create and structure the basic structure of web pages. This is not a programming language, but rather a markup (marker) language that uses certain codes (called tags) so that web browsers can display text, images and other elements. Main FunctionHTML functions as the framework or foundation of a website. You can imagine HTML like a human skeleton, which will then be covered with skin (CSS) and given the ability to move (JavaScript). Specifically, HTML is used to: Display text, paragraphs, and titles (headings). for the title. You open the file in a browser (such as Google Chrome or Mozilla Firefox). The browser will read the HTML code and forward it into a neat visual display that is easy for the user to read.
